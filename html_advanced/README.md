## 📌 À propos

Ce projet couvre les concepts avancés du HTML5 et les meilleures pratiques pour développer des sites web modernes et sémantiquement corrects.

### Objectifs

- ✨ Maîtriser la structure HTML5 sémantique
- 🏗️ Utiliser les bonnes balises pour une meilleure accessibilité
- 📱 Créer des contenus structurés et maintenables
- ✅ Respecter les standards Holberton

## 📂 Structure

```
html_advanced/
├── 0-index.html      # Structure HTML de base
├── 1-index.html      # Ajout lang et dir
├── 2-index.html      # Meta charset, viewport, title, description, favicons
├── 3-index.html      # Header, main, footer
├── article.html      # Page article avec aside
├── 5-index.html      # Sections thématiques
├── 6-index.html      # Articles dans works, news, testimonials
├── 7-index.html      # Navigation dans le header
├── 8-index.html      # Titre h1
├── 9-index.html      # Titres h2 dans chaque section
├── 10-index.html     # Titres h3 dans services, works, about, news
└── README.md         # Cette documentation
```

## 🔍 Vérification
